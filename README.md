# Full-Stack Task Manager (.NET 8 & React 19)

A modern, production-ready Task Management application demonstrating seamless integration between a C#/.Net backend and a React frontend, deployed entirely on the Azure cloud.

**🔗 Live Frontend (Vercel):** https://dot-net-react-todo-app.vercel.app

**🔗 Live API (Azure):** https://rohit-todo-api-d7fnc2fwbda6csdr.centralindia-01.azurewebsites.net/todoitems

## Tech Stack
* **Frontend:** React 19, TypeScript, Vite
* **Backend:** C# .NET 8, ASP.NET Core Minimal APIs
* **Database:** SQLite via Entity Framework (EF) Core
* **Deployment:** Microsoft Azure App Services (CI/CD via GitHub Actions), Vercel

## Architecture Highlights
* Implemented **Minimal APIs** in .NET for high-performance, lightweight routing.
* Utilized **Entity Framework Core** for ORM and database management.
* Configured robust **CORS policies** to securely bridge the Azure API and Vercel frontend.
